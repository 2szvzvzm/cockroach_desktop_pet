# 🪳 蟑螂桌宠

一个基于 Electron 的桌面宠物应用，一只可爱的蟑螂在你的桌面上自由爬行。

## 功能

- **自动爬行**：蟑螂会在屏幕范围内随机移动，包含爬行、暂停、转向等状态
- **拖拽拖放**：按住鼠标左键可以随意拖动蟑螂到任意位置
- **点击互动**：点击蟑螂会触发跳跃、挤压、抖动等动画，并随机弹出对话气泡
- **滚轮缩放**：鼠标滚轮可以放大/缩小蟑螂
- **右键菜单**：支持调整大小（小/中/大）、切换置顶、退出
- **鼠标穿透**：鼠标不在蟑螂身上时自动穿透到桌面，不影响正常操作
- **朝左冲刺**：蟑螂会偶尔快速向左冲刺一段距离

## 运行

```bash
# 安装依赖
npm install

# 启动
npm start
```

## 打包

```bash
npm run build
```

打包输出在 `dist/` 目录下，生成便携版 exe 文件。

## 项目结构

```
roach-pet/
├── main.js          # Electron 主进程
├── index.html       # 渲染进程（蟑螂动画 & 交互逻辑）
├── package.json
└── material/
    └── frameAnimation-moveBack/   # 蟑螂帧动画素材（4 帧）
```

## 技术栈

- Electron
- 原生 CSS 动画 & JavaScript
