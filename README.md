# 🪳 蟑螂桌宠

一个基于 Electron 的桌面宠物应用，一只可爱的蟑螂在你的桌面上自由爬行，还会吃掉你的桌面快捷方式。

## 功能

### 基础互动
- **自动爬行**：蟑螂在屏幕工作区内随机移动，包含爬行、暂停、转向、抖动等状态
- **拖拽拖放**：按住鼠标左键可以随意拖动蟑螂到任意位置
- **点击互动**：点击蟑螂触发跳跃、挤压、抖动等动画，并随机弹出对话气泡
- **滚轮缩放**：鼠标滚轮可以放大/缩小蟑螂（0.3x ~ 3.0x）
- **右键菜单**：调整大小（小/中/大）、切换置顶、退出程序
- **鼠标穿透**：鼠标不在蟑螂身上时自动穿透到桌面，不影响正常操作
- **朝左冲刺**：蟑螂偶尔快速向左冲刺一段距离

### 觅食系统 🍽️
- **觅食模式**：右键菜单开启，蟑螂会自动寻找桌面上的快捷方式（.lnk）并吃掉
- **智能定位**：通过 PowerShell 脚本精确获取桌面图标坐标，蟑螂直奔图标中心
- **咀嚼动画**：到达图标后播放咀嚼帧动画，视觉上"吃掉"快捷方式
- **吞进肚子**：快捷方式文件被移入应用暂存目录（`stomach/`），不会真正删除
- **消化冷却**：吃完一个后有 60~90 秒随机冷却，冷却结束后继续觅食
- **吐出来**：右键菜单可查看肚子里的快捷方式，选择吐回桌面
- **调试红点**：觅食时显示红色标记点，方便验证坐标定位是否准确

## 运行

```bash
# 安装依赖
npm install

# 启动
npm start
```

## 打包

```bash
npm run build
```

打包输出在 `dist/` 目录下，生成便携版 exe 文件（`桌宠 1.0.0.exe`）。

## 技术栈

- **Electron** — 桌面应用框架
- **原生 CSS 动画 & JavaScript** — 帧序列动画、交互状态机
- **透明窗口 + 鼠标穿透** — 不影响桌面正常使用
- **PowerShell + Win32 API** — 跨进程获取桌面图标坐标（`LVM_GETITEMRECT`）
- **IPC 通信** — 主进程与渲染进程数据交互

## 右键菜单

| 菜单项 | 说明 |
|--------|------|
| 调整大小 → 小/中/大 | 缩放蟑螂（0.6x / 1.0x / 1.5x） |
| 开始觅食 | 开关觅食模式，蟑螂自动寻找并吃掉桌面快捷方式 |
| 吐出来 → ... | 查看已吞下的快捷方式，选择吐回桌面 |
| 切换置顶 | 切换窗口是否始终置顶 |
| 退出 | 退出程序 |

## 觅食工作原理

1. 渲染进程请求主进程获取桌面图标列表
2. 主进程调用 PowerShell 脚本，通过 Win32 API（`LVM_GETITEMRECT` + `LVIR_ICON`）获取每个图标的精确中心坐标
3. 坐标经过 DPI 缩放转换后返回渲染进程
4. 蟑螂计算与最近图标的距离，以视觉中心对齐的方式爬向目标
5. 到达后触发咀嚼动画，通过 IPC 将 .lnk 文件移入 `stomach/` 暂存目录
6. 进入消化冷却，冷却结束后继续寻找下一个目标

## 注意事项

- 觅食功能需要 PowerShell 执行权限，首次使用可能被 Windows Defender 拦截
- 打包后 `scripts/` 目录需配置 `asarUnpack` 以确保 PowerShell 脚本可正常调用
- 吃掉的快捷方式存储在 `%APPDATA%/roach-desktop-pet/stomach/` 目录下
