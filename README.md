# 蟑螂桌宠 🪳

一个可爱的 Electron 桌面宠物蟑螂，可以在你的桌面上爬行、觅食、互动。

## 功能特性

- **自主爬行**：蟑螂会在桌面上随机爬行，偶尔停顿、转向、冲刺
- **觅食系统**：开启觅食模式后，蟑螂会寻找并"吃掉"桌面快捷方式图标
- **互动反馈**：点击蟑螂会触发跳跃、抖动等动画，并弹出对话气泡
- **拖拽移动**：可以用鼠标拖拽蟑螂到任意位置
- **滚轮缩放**：鼠标滚轮可以调整蟑螂大小
- **右键菜单**：右键菜单可控制觅食模式开关、查看/吐出被吃掉的图标
- **点击穿透**：蟑螂默认穿透鼠标点击，鼠标进入蟑螂区域时自动接收事件

## 觅食动画

觅食过程包含丰富的动画和气泡反馈：

1. **发现目标**：播放兴奋跳跃动画，气泡显示"哇！好吃的！"
2. **奔向食物**：匀速走向目标图标
3. **咀嚼中**：播放咀嚼帧动画，气泡显示"吧唧吧唧...真香！"
4. **吃完后**：气泡显示"好满足！"，40% 概率触发打嗝动画

## 项目结构

```
roach-pet/
├── main.js                 # Electron 主进程
├── index.html              # 渲染进程（UI + 动画逻辑）
├── package.json            # 项目配置
├── material/               # 动画素材
│   ├── frameAnimation-moveBack/   # 爬行帧动画（4帧）
│   └── frameAnimation-chew/       # 咀嚼帧动画（9帧）
├── scripts/                # PowerShell 脚本
│   └── get-desktop-icons.ps1      # 获取桌面图标坐标
└── dist/                   # 打包输出目录
```

## 运行项目

```bash
# 安装依赖
npm install

# 启动开发模式
npm start

# 打包为可执行文件
npm run build
```

## 技术栈

- **Electron** - 桌面应用框架
- **electron-builder** - 打包工具
- **CSS 动画** - 各种互动动画
- **requestAnimationFrame** - 主循环驱动帧动画和位移

## 配置说明

在 `index.html` 的 `CONFIG` 对象中可以调整：

- `frame.fps` - 爬行帧率
- `crawl.baseSpeed` - 基础爬行速度
- `chew.fps` - 咀嚼帧率
- `chew.duration` - 咀嚼持续时间
- `chew.cooldownSec` - 吃完后的冷却时间
- 更多参数见代码内注释

## 注意事项

- 觅食功能会移动桌面 `.lnk` 快捷方式到应用数据目录，可通过右键菜单恢复
- 打包时 PowerShell 脚本需要放在 `asarUnpack` 中才能被主进程调用
